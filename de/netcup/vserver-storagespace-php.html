Storagespace für vServer / Root-Server

Zu jedem aktuellen vServer / Root-Server können Sie optional Storagespace buchen. Der Storagespace liegt auf einem externen Storage-Verbund und kann gemeinsam, sprich von mehreren Servern und auch von externen Clienten genutzt werden. Der Storagespace wird mittels NFS angesprochen. Die entsprechenden Clients müssen dabei IP-basiert freigeschaltet werden.

Neue Abrechnungsvarianten:

Zur Auswahl stehen mehrere Abrechnungsvarianten. Jede Abrechnungsvariante enthält ein Grundvolumen. Darüber hinaus gehender Speicherplatz wird nach Verbrauch je GB abgerechnet.
